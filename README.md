# ShapeAreaLibrary

**ShapeAreaLibrary** — это библиотека на языке C#, предназначенная для расчета площади различных фигур и проверки свойств треугольников (например, прямоугольные ли они). Библиотека разработана с учетом принципов SOLID и паттернов проектирования для обеспечения расширяемости и простоты использования.

## Основные Возможности

- **Расчет площади круга** по его радиусу.
- **Расчет площади треугольника** по трем сторонам (с использованием формулы Герона).
- **Проверка, является ли треугольник прямоугольным**.
- Легкость добавления новых фигур.
- Расчет площади фигуры без знания типа фигуры на этапе компиляции.
